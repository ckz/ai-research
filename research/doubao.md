## Products & Features

### Core Capabilities
- **Text Chat**: Conversational AI
- **Image Generation**: Text-to-image (Seedream 3.0)
- **Audio Conversations**: Voice interaction
- **Video Creation**: 5-second video generation
- **Multimodal Processing**: Text, image, audio together

### Advanced Features (2025-2026)
- **OS-Level Assistant**: Runs at operating system level on phones
- **Screen Understanding**: Can see what's on screen
- **App Control**: Uses apps and executes tasks
- **Memory**: Stores information (notes, addresses, preferences)
- **Autonomous Tasks**: Ticket bookings, e-commerce integration
- **Douyin Integration**: Deep integration with TikTok China

### Seedream 3.0 (Apr 2025)
- Bilingual text-to-image model (Chinese-English)
- High-resolution output (up to 2K)
- ~3 second generation time
- 94% text rendering accuracy
