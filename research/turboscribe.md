## Products & Features

### Core Technology
- Powered by OpenAI Whisper AI
- 99.8% accuracy rate
- Multi-language support
- Speaker recognition

### Features
- Audio-to-text transcription
- Video-to-text transcription
- Speaker diarization (voice identification)
- Audio restoration tool
- Bulk exports
- Translation to 134+ languages

### Supported Formats
MP3, M4A, MP4, MOV, AAC, WAV, OGG, OPUS, MPEG, WMA, WMV, AVI, FLAC, AIFF, ALAC, 3GP, MKV, WEBM, VOB, RMVB, MTS, TS, QuickTime, DivX

### File Limits
- Free: 30 minutes per file
- Unlimited: 10 hours per file, 5GB max
- Batch upload: 50 files at a time
