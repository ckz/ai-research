## Products & Features

### Core AI Features
- **ChatGPT-4 Enhanced Grammar**: AI-powered proofreading for error-free cross-language writing
- **Document Translation**: 1-click translation of PDF/Word/Excel with original formatting preserved
- **Image Translation**: Real-time text recognition and translation for menus, signs, manuals
- **Offline Mode**: Supported for key features
- **Chat Translation**: Drag-and-translate floating icon for live chat/social media

### Language Support
137 languages including:
- English, Arabic, Hindi, Spanish, French
- Japanese, Korean, Turkish, Urdu, Farsi
- Vietnamese and many more

### Smart Scene Learning
- Chat scenarios
- Work scenarios
- Travel scenarios
- Study scenarios
